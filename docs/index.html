What is VSP-VOIP?

VSP-VOIP is a RingCentral-style multi-tenant business phone platform. A single platform operator (Super Admin) provisions companies (Tenants), each with phone numbers, users, call routing, SMS, voicemail, recordings, and WebRTC softphones. Tenants use a Next.js web portal and a Flutter mobile app; both talk to the same Express REST API on port 3000. Voice and messaging are delivered through Telnyx; number purchases and subscriptions use Stripe.

Quick Start

  • API: npm run dev:api (port 3000)
  • Web: npm run dev:web (port 3001)
  • Seed: npm run seed — creates demo tenant + super admin
  • Mobile: cd mobile && flutter run --dart-define=API_BASE_URL=http://10.0.2.2:3000
