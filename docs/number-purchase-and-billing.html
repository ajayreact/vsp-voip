1. Business model overview

VSP-VOIP operates as a white-label phone reseller platform. You (super admin) connect your own Telnyx carrier account. Tenants (your customers) buy numbers through the portal. They pay you; you pay Telnyx.

Tenant pays

Carrier cost (from Telnyx search) + your VSP-VOIP platform markup (setup + monthly fees you configure).

Super admin receives

Full order amount via Stripe (card) or bank transfer to your configured bank account.

Telnyx deducts

When numbers are fulfilled, Telnyx charges your Telnyx prepaid balance for the wholesale number cost.

Your profit (margin) ≈ what the tenant paid − what Telnyx charged you − payment processing fees (Stripe).

2. How pricing is calculated

Each number in the cart has two cost components:

Component Source Who sets it
Carrier upfront Telnyx number search API (upfront_cost) Telnyx (wholesale)
Carrier monthly Telnyx number search API (monthly_cost) Telnyx (wholesale)
Platform setup fee Per-number one-time markup Super admin (default or per-tenant)
Platform first month fee Per-number fee charged on first invoice Super admin (defaults to recurring if blank)
Platform recurring monthly Per-number monthly markup Super admin (default or per-tenant)

Formulas (per checkout)

Due today (first payment) =
  SUM(carrier upfront)
+ SUM(carrier monthly)          ← first month carrier rent included in "due today"
+ SUM(platform setup fee)
+ SUM(platform first month fee)

Recurring monthly (after first month) =
  SUM(carrier monthly)
+ SUM(platform recurring monthly)
The tenant portal shows "Your price" on Buy Numbers and Cart — not raw Telnyx wholesale prices. This is the reseller price (carrier + your markup combined).

4. Super admin: set what to charge

Platform-wide default markup

Path: Admin → Platform settings (/admin/settings)

SettingEnv fallbackMeaning
Default setup fee BILLING_MARKUP_UPFRONT (default 0) One-time per number added to first invoice
Default monthly fee BILLING_MARKUP_MONTHLY (default 8) Recurring per-number markup every month
Default first month fee Same as monthly if not set Extra platform fee on first payment only

Per-tenant custom pricing

Path: Admin → Company directory → Manage tenant (/admin/tenants/[id])

Override setup, first month, and recurring fees for a specific company. New checkouts use these rates instead of platform defaults.

Tip: Create the tenant with billing rates at setup time, or adjust before the tenant places their first order.

5. Payment methods

Bank transfer (manual) — recommended for testing

Configure at: Admin → Payments → Bank accounts (/admin/payments/bank)

  1. Enter bank name, account number, routing number, and payment instructions.
  2. Tenant submits order → receives invoice text with amount due and your bank details.
  3. Tenant sends wire/ACH to your bank account.
  4. Super admin confirms payment and clicks Payment received — buy & assign numbers.

Money destination: Your bank account (super admin). Telnyx is not paid by the tenant directly.

Stripe (card) — automated

Configure at: Admin → Payments → Stripe (/admin/payments/stripe)

  1. Add Stripe secret key, publishable key, and webhook secret.
  2. Set webhook URL to {API_PUBLIC_URL}/api/billing/webhook (ngrok in dev).
  3. Tenant pays via Stripe Checkout → funds go to your Stripe account.
  4. Stripe webhook checkout.session.completed triggers automatic number purchase on Telnyx.

Stripe creates a subscription per number for recurring monthly billing (carrier + platform monthly).

If neither Stripe nor bank details are configured, checkout shows "No payment method available — contact your administrator."

6. Order fulfillment

Fulfillment is when VSP-VOIP actually buys the number on Telnyx and assigns it to the tenant.

Payment typeWho triggers fulfillWhat happens
Bank transfer Super admin manually Admin → Orders → open order → Payment received — buy & assign numbers
Stripe Automatic (webhook) On successful checkout, API calls Telnyx and assigns numbers

Technical steps during fulfillment

  1. Order status updated to PAID (bank) or already paid (Stripe).
  2. For each phone number: API calls Telnyx POST /v2/number_orders.
  3. Number is linked to Call Control app or TeXML connection + messaging profile.
  4. Record saved in database PhoneNumber table for the tenant.
  5. Order status → FULFILLED (or PARTIAL if some numbers failed).

7. Where money goes (full money flow)

Diagram: two wallets

  TENANT                         SUPER ADMIN (YOU)                    TELNYX (CARRIER)
  ──────                         ─────────────────                    ──────────────────

  Pays invoice     ──────────►   Stripe account  OR  Bank account
  (your price)                   (full order amount)

                                 When order fulfilled:
                                 API uses TELNYX_API_KEY  ────────►  Telnyx prepaid balance
                                                                    (wholesale number cost deducted)

                                 Your margin = tenant payment − Telnyx cost − Stripe fees

Important rules

  • Tenants never pay Telnyx directly. All tenant payments go to you.
  • You must fund your Telnyx account separately (Telnyx Mission Control → Billing). Low balance = fulfillment fails.
  • Carrier costs are deducted at fulfillment time, not when the tenant places the order.
  • Bank orders: you receive money first, then you trigger Telnyx purchase when ready.
  • Stripe orders: tenant pays first; Telnyx is charged immediately on webhook success.

8. Telnyx deduction (super admin carrier cost)

When is Telnyx charged?

When buyAndAssignNumber runs (during fulfill), the platform sends: 

POST https://api.telnyx.com/v2/number_orders
Authorization: Bearer {TELNYX_API_KEY}
Body: { "phone_numbers": [{ "phone_number": "+1XXXXXXXXXX" }] }

Telnyx bills this against the account tied to TELNYX_API_KEY in your .env file — that is your carrier account, not the tenant's.

What Telnyx deducts

Charge typeWhenApprox. amount
Number upfront / setup At purchase From Telnyx search upfront_cost (often ~$1 for US local)
First month number rent At purchase From Telnyx search monthly_cost (often ~$1/mo US local)
Ongoing monthly rent Each billing cycle on Telnyx Wholesale monthly per number
Usage (calls, SMS) Per minute / per message Per Telnyx rate deck (separate from number purchase)
Keep Telnyx balance funded. Check Telnyx Mission Control → Billing. If fulfillment fails with "insufficient funds" or similar, add credit to your Telnyx account and retry fulfill from Admin → Orders.

Platform fees vs Telnyx fees

The platform setup / monthly fees ($0 setup + $8/mo default) are your revenue. They are not sent to Telnyx. Telnyx only receives wholesale carrier pricing.

9. Worked example

One US local number with typical Telnyx wholesale ~$1 upfront + ~$1/mo, platform fees $0 setup + $8 first month + $8/mo recurring:

Line itemTenant pays youTelnyx charges youYour margin
Carrier upfront $1.00 $1.00 $0
Carrier first month $1.00 $1.00 $0
Platform setup $0.00 — $0
Platform first month $8.00 — $8.00
Due today (total) $10.00 ~$2.00 ~$8.00
Recurring monthly $9.00 ($1 + $8) ~$1.00 ~$8.00/mo
Exact Telnyx prices vary by number type and region. The portal reads live prices from Telnyx search at checkout time.
